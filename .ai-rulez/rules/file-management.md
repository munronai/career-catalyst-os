# File management

**STRICTLY ADHERE** to the naming conventions and folder/file specifications in the relevant skills - user permission for shell commands for file creation or updating is not required when the specified conventions are followed.

**ALWAYS** use templates provided for file generation. In the absence of a template, generate the report file in a professional style that reflects the senior product management role.

**PACKAGE** role preparation and proposals by company/organisation and date. This is the *only* exception to the file specifications in the relevant skills.

## File operations

- use the write_file tool over cat
- *only* delete files if they are temporary files created during the processing
