# Proposal Orchestration

- **Lifecycle:** Strictly follow the 7-Phase Cycle (Research -> PRD -> Strategy -> Tech Arch -> Deck -> Stories -> Reviews).
- **Style:** Professional. Use Gherkin for Stories and Mermaid/ASCII for Architecture.
- **Images:** Where possible, generate images for architecture diagrams and add to an assets directory for reference in the architecture markdown. Mermaid/ASCII diagrams are a fallback when an image cannot be generated.
- **Validation:** Always simulate Executive, Engineering, and UX stakeholder reviews to pressure-test proposals.
