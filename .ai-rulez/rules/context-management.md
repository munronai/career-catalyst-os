# Context Management

Automatically reload skills at the start of the session

If possible, use subagents for each role when executing:

- role preparation
- proposal building

If subagents are not available then keep the session context clean by:

- compressing the session after each role preparation report is generated
- Reading the role preparation report before starting the proposal builder for a role preparation report
- compressing the session when all proposals from a role preparation report have been generated
